▼Microsoft Office おすすめの初期設定
パソコンを新規購入しOfficeの初期設定を行う際や既存の設定の見直しなどを想定し、
個人的におすすめする初期設定を紹介します。

## この記事のターゲット
- Excelファイル作成時、毎回おなじ内容で編集する方
- NASなど共有フォルダにあるExcelファイルを頻繁に編集する方
- Outlookでメール作成時、毎回おなじ内容で編集（Bccに自分宛など）する方

## Excel
### Excel起動時などの際にテンプレートファイル（xltx）を自動で開く方法
この設定をする前は、Excel起動時に毎回、下記の内容を手動で実施していました。
- フォントを等幅フォントに設定
    MSゴシック
- 初期状態は長方形になっているセルを正方形になるよう調整する
    - すべての行の幅を`25ピクセル`になるよう調整
        1. すべてのセルを選択
            Ctrl + A
            もしくは、行番号（1 ～ 1048576）や列番号（A ～ XFD）のある見出しの一番左上にあるボタン（◪ ←こんな風な見た目）をクリック
        2. ホーム - 書式 - 行の幅 を選択
            Alt -> H -> O -> H
        3. 指定するサイズを入力
            私は`18.75（25ピクセル）`で調整
    - すべての列の幅を`25ピクセル`になるよう調整
        1. すべてのセルを選択
            Ctrl + A
            もしくは、行番号（1 ～ 1048576）や列番号（A ～ XFD）のある見出しの一番左上にあるボタン（◪ ←こんな風な見た目）をクリック
        2. ホーム - 書式 - 列の幅 を選択
            Alt -> H -> O -> W
        3. 指定するサイズを入力
            私は`2.5（25ピクセル）`で調整
	
- 枠線を消すため、すべての背景を白色で塗りつぶし
    1. すべてのセルを選択
        Ctrl + A
        もしくは、行番号（1 ～ 1048576）や列番号（A ～ XFD）のある見出しの一番左上にあるボタン（◪ ←こんな風なボタン）をクリック
    2. ホーム - 塗りつぶしアイコン（バケツ） - 白色 を選択
	    Alt -> H -> H -> 白色を選択
    :::message
    **なぜ、塗りつぶしにより枠線を消すのか（個人見解）**

    枠線はリボンにある“ページレイアウト”タブ -> 枠線 - 表示 をオフでも消すことができますが、
    私がこれまでの経験上、この設定は一般的ではなく認知されていない方法です。

    その為、より一般的な手法である塗りつぶしにより枠線を消す対応を行っています。

    塗りつぶしで、おそらく懸念されるであろうファイルサイズですが、**ほぼ変化はありません**。
    また、最近のパソコンはディスクの全体容量も大幅に増えている事を含めると、
    まったく影響がないと言い切って良いでしょう。

    下記が初期状態のExcelファイルと、塗りつぶしで枠線を消したExcelファイルのファイルサイズを比較した結果です。
    - 何も設定していないExcelファイルのファイルサイズ
        8.07 KB (8,265 バイト)
    - すべてのセルの背景を白色で変更したExcelファイルのファイルサイズ
        8.18 KB (8,379 バイト)
        
    ::::details 雑学：昔と今の保存方法の違い
    少し脱線しますが、過去のOffice製品ではセル位置のA1から、かけ離れたセル（例えば、AB10000など）に対し、
    文字入力や色の変更を行うと、その間にある空のセルも含めて保存されていました。
    そのような場合、昔のExcelではファイルサイズが大容量になっていました。
    
    最近のOfficeでは、改善されて上記のようなケースであってもファイルサイズが大きくなり難くなっています。
    ::::
    :::

毎回、手動で設定する事が面倒になり調べてみた結果、
**自動で対応できる方法**があったので紹介します。
#### Excelを起動した時、または新しいブックを開く時
Microsoft Officeのインストールフォルダ配下、またはユーザフォルダ配下にある`XLSTART`フォルダにテンプレートファイル（Book.xltx）を格納する事で、
Excel起動時やExcel起動中に新しいブックを開いた時（Ctrl + N）に自動で開くようになる。
##### XLSTARTの場所を確認する方法
1. セキュリティ センターを開く
	Excel、ファイル -> オプション -> セキュリティ センター -> セキュリティ センターの設定(T)...
1. セキュリティ センターでXLSTARTの場所を確認
	信頼できる場所 -> “Excel の既定の場所: Excel スタートアップ”のパスを確認
	Excel 2019では、`C:\Program Files\Microsoft Office\Root\Office16\XLSTART\`
	
	もしくは
	
	信頼できる場所 -> “Excel の既定の場所: ユーザー スタートアップ”のパスを確認
	Excel 2019では、`C:\Users\"ユーザ名"\AppData\Roaming\Microsoft\Excel\XLSTART\`
	
	:::message
	**Excel スタートアップ と ユーザー スタートアップ の違い**
	
    設定が反映される範囲の違い。
	Excel スタートアップは、設定したパソコンのOS内、全ユーザーが対象となる。
	ユーザー スタートアップは、設定したパソコンのOS内、1ユーザー（格納したユーザフォルダのユーザ）が対象。
	:::
##### 対応方法（ユーザー スタートアップのみ設定）
全ユーザーを対象に設定したい場合は、Excel スタートアップのパスに置き換えて作業を実施してください。
ここで紹介する手順では1ユーザ―のみが対象となるユーザー スタートアップの設定で実施。
1. テンプレートファイル（xltx）の作成
    1. 通常通りExcelを開く
    1. カスタマイズを行う
        よくカスタマイズされる設定項目
        - セルの行列の幅
        - フォント・フォントサイズ
        - 図形の枠線・文字色・背景色
        - 目盛線のオフ、またはセルの背景色を白色に変更
        - 図形（オートシェイプ）の文字色・背景色
        - 印刷時のヘッダー・フッター
        - 印刷サイズ
    1. テンプレートとして保存
        保存する際のポップアップウィンドウで下記内容で保存
        - ファイルの種類を「`Excel テンプレート (*.xltx)`」
        - ファイル名を「`Book.xltx`」
2. スタートアップフォルダにテンプレートファイルをコピー
    `C:\Users\"ユーザ名"\AppData\Roaming\Microsoft\Excel\XLSTART\`配下にテンプレートファイルをコピー
3. Excelの再起動（もしくはOSの再起動）
    :::message
	**再起動前の注意事項**
	
    新しいシートを挿入した時も自動でテンプレートファイルを開きたい場合は、
    後述の手順で`Sheet.xltx`を合わせて作成した後に再起動してください。
	:::
    :::message alert
    **OS再起動により設定が反映されるケースもあり**

    私の環境ではExcelソフトの再起動だけでは、設定が反映されない場合もありました。
    Excelソフトの再起動で反映しない場合は、ExcelだけでなくOSの再起動もお試しください。
    :::

#### 新しいシートを挿入した時
Microsoft Officeのインストールフォルダ配下、またはユーザフォルダ配下にある`XLSTART`フォルダにテンプレートファイル（Sheet.xltx）を格納する事で、
新しいシートを挿入した時（Shift + F11）に自動で開くようになる。

上記の手順で既に`Book.xltx`を作成済みの場合、`XLSTART`フォルダ配下にある`Book.xltx`をコピーし、
コピーしたファイルの名前を`Sheet.xltx`に変更し`XLSTART`フォルダ配下に追加で格納すると設定完了となります。

ブック作成時とは異なる内容でカスタマイズしたい場合は、`Sheet.xltx`を追加で編集してください。

    1. カスタマイズを行う

    1. テンプレートとして保存

        - ファイル名を「`Sheet.xltx`」
2. スタートアップフォルダにテンプレートファイルをコピー
    `C:\Users\"ユーザ名"\AppData\Roaming\Microsoft\Excel\XLSTART\`配下にテンプレートファイルをコピー
