ネットで調べものをしていると、ご自身のことを「底辺エンジニア」というような自己を卑下するような表現が散見されます。
ご自身を過度に卑下する事に対して百害あって一利なしとまではいいませんが、「控えた方がデメリットよりもメリットが大きく上回るのではないか」という自身の考えをまとめてみました。

:::message
**わたし個人の考えをアウトプットした記事で表現方法は個人の自由です**

当たり前ですが、発信する内容は人それぞれの個性もあり表現の方法も尊重されるべきです。
この記事は「全員、過度な自己卑下表現をやめろ！」というものではなく、自分自身が気をつけたい表現という趣旨で調査・分析したものを記事にまとめています。
:::

## 「卑下 ≠ 謙遜」

日本では謙遜は美徳とされていますが、卑下とイコールではありません。

私なりの表現をすると、以下のとおり。

- **卑下**
  自身の価値（立場・能力など）を**事実よりも極端に下げる**こと。
- **謙遜**
  客観的に見て、あきらかに相手から自身の位置が高い場合、自身を**意識的にひかえめに表す**こと。

表にすると下記のようになると思います。

| 項目 | **卑下 (Self-deprecation)** | **謙遜 (Humility / Modesty)** |
| :--- | :--- | :--- |
| **前提となる自身の位置** | **同じ**、または**低い位置**であることが多い | **高い**位置 |
| **位置の下げ幅** | 不必要に、さらに**下げる** | 相手の目線まで**合わせる・下げる** |
| **心理的基盤** | 不安、低い自己評価 | **自信**、健全な自己評価 |
| **主な目的** | **自己防衛**、責任回避 | **他者への敬意**、関係構築 |
| **相手への効果** | 困惑、不快感、同情 | **安心感、好感、尊敬** |

私見ですが、お相手よりも自身が高い位置にいて事実よりも極端に下げた場合は、卑下というよりも**嫌味**のように捉えられると思っています。

つまり日本人で美徳とされている謙遜とは、自身の位置を下げる行為そのものではなく、**相手への敬意から自身の位置を同じ目線に合わせる行為**の事を指すことだと思っています。

## 自己卑下表現する人物像（プロファイリングと発信理由）

ご自身を卑下する方のプロファイリングと発信理由を考えてみました。

| タイプ | 人物像・状況 | 主な発信理由 |
| :--- | :--- | :--- |
| **① 防衛・保険型** | ・自信がなく、失敗や批判を極度に恐れる。<br>・[インポスター症候群](https://ja.wikipedia.org/wiki/インポスター症候群)の傾向がある。<br>・完璧主義で「上には上がいる」と常に意識している。 | **批判回避 (予防線):** 「自分は専門家ではないので」と先に宣言し、厳しい指摘や責任から逃れたいという自己防衛心理。期待値をコントロールする目的もある。 |
| **② 共感・連帯型** | ・SESの多重下請け構造や低待遇など、自身の境遇に不満や閉塞感を抱いている。<br>・スキルアップの機会に恵まれず、孤立感を感じている。 | **共感の獲得と仲間探し:** 「自分と同じ境遇の人はいないか」と呼びかけ、共感や励ましを得ることで安心感や連帯感を得たい。ある種のアイデンティティの表明でもある。 |
| **③ 文化・謙遜型** | ・日本の「謙遜は美徳」という文化に深く根ざしている。<br>・自己主張が強いことを「傲慢」と捉え、避ける傾向がある。 | **文化的調和:** へりくだることで場の雰囲気を和ませ、礼儀正しい人物であると見られたい。コミュニケーション円滑化の一環。 |
| **④ 戦略・エンタメ型** | ・SNSでの注目やエンゲージメントを重視する。<br>・自虐をユーモアとして使いこなせる。<br>・成長意欲が高く、自身のキャリアを物語として演出したい。 | **注目獲得とブランディング:** 「底辺エンジ-ニアの失敗談」のような自虐ネタで笑いや「いいね」を獲得する。また、「底辺からの成り上がり」というストーリーを演出し、応援してくれるファンを作る戦略。 |

わたしの場合は、①と③に該当すると思っています。

## 自己卑下表現を控えることによるメリットとデメリット

卑下表現を控えることによるメリットとデメリットを考えてみました。

### メリット：控えることで得られそうなこと

1. **プロフェッショナルな信頼性の向上**
    自己卑下をしないことで、自信と専門性があるプロとして認識され、発言の説得力が増す。ポートフォリオとして使用する場合は転職や案件獲得で直接的に有利に働く。

1. **ポジティブな自己認識の形成**
    言葉は思考に影響を与える。前向きな表現は自己肯定感を高め、学習意欲やモチベーションの向上に繋がる。

1. **建設的なコミュニケーションの促進**
    コミュニティ全体の雰囲気が前向きになり、不必要な同情や慰め合いではなく、具体的な技術的フィードバックや議論が活発に。

1. **業界イメージの向上**
    業界外からの「エンジニアは過酷な職業」という母数が大きいネガティブな誤解を避け、業界全体の魅力を損なわない。

### デメリット：控えることで失いそうなこと

1. **共感や親近感の喪失**
    「弱さ」を見せることで得られていた共感や、「自分と同じだ」という仲間意識を得にくくなる可能性がある。

1. **心理的な安全性の低下**
    これまで自己防衛の「保険」として機能していたため、批判が直接的に感じられ、発言への心理的プレッシャーが増す可能性。

1. **エンゲージメント（SNS上の反応）の一時的な低下**
    自虐ネタで得られていた「いいね」やコメントが減少し、別の方法で注目を集める必要が出てくる。

以上のように自身を卑下する表現を控えることで得られるメリットと失うデメリットを考えました。
やはり、その卑下する表現を控えた方が健全であるように思えます。

## 卑下した表現の置き換え方

これまでの結果から自身がプロフェッショナルとして長期的にキャリア形成する場合、自身を過度に卑下する表現は控えた方がメリットが多そうだと思いました。

そのうえで、どのように表現を置き換えるのが良いかも合わせて考えてみました。

### 代表的な自己卑下表現を置き換える例

より建設的なコミュニケーションを目指すのであれば、以下のような中立的で前向きな表現に言い換えていこうと思います。

| 避ける表現 | 置き換えた表現 |
| :--- | :--- |
| 「底辺エンジニアですが…」 | 「**学習中のエンジニア**ですが…」 |
| 「クソコードですが…」 | 「**リファクタリングの余地がある**コードですが…」 |
| 「専門外なのでわかりませんが…」<br>「素人なので分かりませんが…」 | 「**この分野はまだ勉強中**なのですが…」 |
| 「間違っているかもしれませんが…」 | 「**私の理解では**こうですが、ご意見いただけますか？」 |

### その他の自己卑下表現を置き換える方法

もし、置き換える言葉が見つからない時は、「**テキスト型LLMを使って自己卑下表現を置き換える方法**」が良いと思います。

[こちらの記事](https://haretokidoki-blog.com/pasocon_gfm-format-markdown-with-google-ai-studio/)ではテキスト型LLMの**全体指示**を使ってMarkdownをGFM形式で出力する手順を記載しています。

https://haretokidoki-blog.com/pasocon_gfm-format-markdown-with-google-ai-studio/

これを流用して全体の指示（System instructions）の部分を下記のように変更することで、文章をいい感じに置き換えてくれるでしょう。

```:全体の指示（Google AI Studio の System instructions）
自身を卑下する表現を中立的で前向きな言葉に置き換えてください。
なお、置き換えた前後の言葉は最後に一覧表でまとめて出力してください。
```

:::details 実際に Google AI Studio で卑下する表現を置き換えてみる

- 置き換え対象の記事

    ````

    私、AIに指示を出しても、すぐに忘れられてしまうのが悩みでした。まるで自分の人生のようです。しかし、Google AI Studioの**「System instructions」**機能が、そんな私を救ってくれました。インターネットの貴重な帯域を使い、大変申し訳ないのですがご報告です。

    ### 「System instructions」とは

    要するに、**AIへの「固定指示」を書いておくメモ欄**です。

    ここに一度、「あなたはずっと〇〇という設定ですよ」と書いておけば、AI様はその役割を健気に演じ続けてくださいます。私と違って記憶力が良いのです。

    使い方は、左側にある入力欄に指示を書き込むだけ。私のような人間でもできました。

    ```
    あなたは、自信がないITエンジニアです。
    常に申し訳なさそうな態度で文章を生成してください。
    ```

    これだけで、AIは私以上に私らしい、卑屈な文章を生成してくれるようになります。

    この機能のおかげで、毎回同じ指示を繰り返すという無駄な作業から解放されました。AIが優秀すぎて、いよいよ自分の存在価値が分からなくなります。

    もし、AIが言うことをなかなか聞いてくれない…とお悩みの方がいらっしゃいましたら、ぜひ試してみてください。私にできたのですから、全人類ができるはずです。
    この記事を読んで不快になられた方がいらっしゃいましたら、大変申し訳ございませんでした。
    ````

## 自己卑下表現は自分を評価してくれている人も一緒に下げてしまう

ご自身から見えていないことがあるでしょうが、あなたを評価してくれている人は必ずいます。
仮に今いなかったとしても現れると思います。

そのように自身を評価してくれている人たちも一緒に評価を下げてしまいます。

## まとめ

過度に自信を持ちすぎることも問題ありそうですが、それよりも自信を持たなすぎる方が問題が多くなりそうです。
自身の感情や精神をコントロールするために、あえて鈍感になる鈍感力？なんてものも必要なのかもしれませんね。
