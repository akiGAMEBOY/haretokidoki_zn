たとえば、ホスト名 と ポート番号 のデータが格納されている2次元配列を繰り返し実行して、
ポートの疎通確認を行う場合、

```powershell:ポートの疎通確認
$port_check_lists = @(
    @('localhost','8080'),
    @('localhost','8000'),
    @('google.com','443'),
    @('smtp.google.com','25')
)

for($i = 0; $i -lt $port_check_lists.Count; $i++) {
    $port_check_command = 'Test-NetConnection '
    $target_row = $port_check_lists[$i]

    Test-NetConnection $target_row[0] -Port $target_row[1]
}
```
