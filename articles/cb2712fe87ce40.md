```powershell:
$rootFolder = 'C:\E-DataPackager\input'
$csvPath = "$($rootFolder)\HeaderValues.csv"

$inputArray = Import-Csv -Path $csvPath
```

- ヘッダーあり ＋ データ0行
    .Count → 0件
- ヘッダーあり ＋ データ1行
    .Count → 結果なし
- ヘッダーあり ＋ データ2行
    .Count → 2件
- ヘッダーなし ＋ データ0行 （コマンドでヘッダーを指定した場合）
    .Count → 結果なし

なぜかデータ1行でImport-Csvで取り込み、Countしても結果が返ってこない。
