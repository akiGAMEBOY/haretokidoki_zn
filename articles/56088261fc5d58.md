winget list で確認するとソースが空欄となっている ＋ スタートメニューにあるアプリを右クリックでStoreアプリ のものを
洗い出し、
すべてwinget install xxx --source msstore でインストールする事で、winget upgrade -all で一括でアップグレード・アップデートが可能となる。
