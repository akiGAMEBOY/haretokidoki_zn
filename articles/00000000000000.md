## Excel
### Excel起動時にExcelテンプレートファイル（xltx）を自動的に開く方法
この設定をする前は、Excel起動時に毎回、下記の対応を行っていました。
- フォントを等幅フォントに設定
    MSゴシック
- 間延びしている列の幅を小さくする
	具体的にはすべての列を行の幅`25ピクセル`に合わせて設定。
	すべて選択（Ctrl + A） -> ホーム - 書式 - 列の幅（Alt -> H -> O -> W） -> `2.5（25ピクセル）`を入力
- 枠線を消す為、すべての背景を白色で塗りつぶし
	すべて選択（Ctrl + A） -> ホーム - 塗りつぶしアイコン（バケツ） - 白色を選択（Alt -> H -> H -> 白色を選択）

毎回、手動で対応する事がオックウになり調べてみた結果、
**自動で対応できる方法**があったので紹介します。
#### Excelソフトを起動した時／新しいブックを開く時
Microsoft Officeのインストールフォルダ配下、またはユーザフォルダ配下にある`XLSTART`フォルダにテンプレートファイル（xltx）を格納する事で、
Excel起動時や新しいブックを開いた時（Ctrl + N）に自動で開くようになる。
:::message
**設定後の注意事項**

私の環境では、OSの再起動後に設定が反映されました。
Excelソフトの再起動で反映されないという方は、OSの再起動もお試しください。
:::
##### XLSTARTの場所を確認する方法
1. セキュリティ センターを開く
	Excel、ファイル -> オプション -> セキュリティ センター -> セキュリティ センターの設定(T)...
1. セキュリティ センターでXLSTARTの場所を確認
	信頼できる場所 -> “Excel の既定の場所: Excel スタートアップ”のパスを確認
	Excel 2019では、`C:\Program Files\Microsoft Office\Root\Office16\XLSTART\`
	
	もしくは
	
	信頼できる場所 -> “Excel の既定の場所: ユーザー スタートアップ”のパスを確認
	Excel 2019では、`C:\Users\"ユーザ名"\AppData\Roaming\Microsoft\Excel\XLSTART\`
	
	:::message
	**Excel スタートアップ と ユーザー スタートアップ の違い**
	
    設定を反映する範囲が違う。
	Excel スタートアップは、設定したパソコンのOS内、全ユーザーが対象
	ユーザー スタートアップは、設定したパソコンのOS内、1ユーザー（格納したユーザフォルダ）が対象
	:::
##### 対応手順
1. テンプレートファイル（xltx）の作成
    1. 通常通りExcelを開く
    1. カスタマイズを行う
        主にカスタマイズするであろう項目
        - フォント・フォントサイズ
        - 図形の枠線・文字色・背景色
        - 目盛線のオフ、またはセルの背景色を白色に変更
        - 図形（オートシェイプ）の文字色・背景色
        - 印刷時のヘッダー・フッター
        - 印刷サイズ
    1. テンプレートとして保存
        保存する際のポップアップウィンドウで下記内容で保存
        - ファイルの種類を「`Excel テンプレート (*.xltx)`」
        - ファイル名を「`Book.xltx`」
2. スタートアップフォルダにテンプレートファイルをコピー
    `C:\Users\"ユーザ名"\AppData\Roaming\Microsoft\Excel\XLSTART\`配下にテンプレートファイルをコピー
3. OS再起動
    :::message
	**OS再起動前の注意事項**
	
    新しいシートを新規挿入した時もテンプレートファイルを開きたい場合は、
    後述の手順で`Sheet.xltx`を合わせて作成した後にOS再起動してください。
	:::

#### 新しいシートを新規挿入（Shift + F11）した時
上記の手順で既に`Book.xltx`を作成済みの場合は、`C:\Users\"ユーザ名"\AppData\Roaming\Microsoft\Excel\XLSTART\`配下で、
`Book.xltx`をコピーし名前を`Sheet.xltx`に変更すると簡単に設定できます。
